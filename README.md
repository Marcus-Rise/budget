# Budget

## Requirements

- `yarn`

## Preparation

- Copy `.env.example` as `.env`
- Pass `API_URL` value (project: https://github.com/Marcus-Rise/budget-api)

## Local run

- Install dependencies

```shell
yarn install
```

- Run local dev server

```shell
yarn dev
```

## Storybook

```shell
yarn storybook
```
